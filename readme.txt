TFM_DSC_Easy_Money Project
--------------------------

Para mi trabajao final de máster, me sumergí en el papel de un científico de datos que trabaja para una empresa que comercializa productos financieros. 
Lo hice para simular un escenario de trabajo real, en el que los requisitos pueden no estar claramente definidos y tenemos que usar nuestra creatividad para idear soluciones.

Ficheros:

Pre-Processing:

	* TFM_DSC_Easy_Money_Cleaning.ipynb: Data Cleaning, análisis del target y analásis de variables respecto al target
	

Modelos de Prospección:

	Consta de 4 notebooks con cuatro modelos distintos entrenados. Finalmente nos quedamos con el útlimo, en el que hemos aplicado un oversamplig.

	Al final de cada notebook están las conclusiones a las que hemos llegado para cada uno:

		* TFM_DSC_Easy_Money_MODELO_1

		* TFM_DSC_Easy_Money_MODELO_Gradient Boosting_2

		* TFM_DSC_Easy_Money_MODELO_UNDERSAMPLING_3

		* TFM_DSC_Easy_Money_MODELO_OVERSAMPLING_4
		
		* TFM_DSC_Easy_Money_MODELO_Pension_plan
		
Clustering:

	* TFM_DSC_Easy_Money_Cleaning_revisado
